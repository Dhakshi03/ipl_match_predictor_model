# Phase 2: Comprehensive ML Model Training Plan

## Objective
Train 10 ML models with multiple hyperparameter configurations using cross-validation, log everything to MLflow, and generate a detailed performance analysis report.

---

---

## Cross-Validation Strategy
- **5-Fold Stratified Cross-Validation** on training data
- Ensures equal team representation in each fold
- Log: mean accuracy, std, per-fold scores

---

## Preprocessing
1. **Feature Scaling**: StandardScaler for SVM, MLP, Logistic Regression, Ridge
2. **Class Weights**: 'balanced' for all models to handle 11-class imbalance
3. **No scaling needed for**: Tree-based models (RF, XGBoost, AdaBoost, GB, CatBoost)

---

## MLflow Logging
For each run, log:
- Parameters (hyperparameters)
- Metrics: accuracy, precision, recall, f1 (both CV and test)
- Model artifact
- Tags: model_type, configuration_name
- Notes: feature scaling applied (yes/no)

---

## Post-Training Analysis
1. Compare all 30 configurations
2. Identify best overall model
3. Create Hard Voting Ensemble from top 5 models
4. Create Soft Voting Ensemble from top 5 models
5. Generate detailed analysis report

---

## Expected Improvements
- CatBoost: Native categorical handling
- Class weights: Better prediction for minority teams
- CV: More robust accuracy estimate
- Ensemble: Potentially higher than any single model
